- osc_deckControl_Example.toe:

This is a touch designer example for how to format OSC messages to enable
control of smart clips in the performer mode. This can be a reference for other OSC enabled devices on how to format the names and values.

-----

- teensy_serial_install.exe:

If you plan to use a teensy 3.1(arduino) as a pixel controller be sure to install it's serial usb driver or GeoPix will not be able to communicate with it correctly. Windows 8/10 usually has the neccesary drivers already installed so this step may not be neccesary.
